# デザイン野帳#3：エナクティブ認知 — 表象を外したとき、認知はどこに立つか

### この本の読み方の補助線 — Andy Clark との対比で読む

vol 02 では、4E のうち身体性・埋め込み・拡張性の三つを追った。今回は最後の一つ、**エナクティブ（行為志向的）**に入る。

ギャラガーは本書で、Andy Clark を章を横断して何度も引き合いに出す。拡張心の提唱者として、予測的処理（PP）能動推論派の代表として、エナクティブ派が越える「最後の壁」の体現者として、自由エネルギー原理（FEP）論争で自ら譲歩する論者として。Clark を foil（引き立て役）として置きながら読むと、本書の語り口がぐっと立体的に見えてくる。

Clark は David Chalmers との共著論文（1998）で拡張心仮説を提唱した一人で、日本でも『現れる存在』『生まれながらのサイボーグ』で知られる 4E の中心的論者だ。書き手の読み筋を率直に言えば、Clark は「最も洗練された 4E 理論家、**しかし最後の壁の手前で立ち止まる人**」。Clark がどこまで来て、どこで踏み止まるのか——その線を見ることで、エナクティブ派が何を踏み越えたのかが鮮明になる。

二つの対立線を引いておく。**機能主義の壁** — Clark は CC（古典的認知主義）から意外なほど近く、機能主義と「行為志向表象（AOR）」を最後まで残した。**系譜の違い** — Clark の機能主義・PP が英米分析哲学の系譜にあるとすれば、エナクティブ派はフッサール・メルロ＝ポンティ・ヴァレラという**現象学**の系譜をそのまま認知科学に持ち込んでいる。vol 01 で「ギャラガーは現象学と認知科学を横断する哲学者」と紹介したのは、まさにこの位置取りのことだ。

エナクティブは、認知を**脳・身体・環境が動的に統合された一つの系**として捉え直し、その統合の中身を機能主義ではなく生命と現象学の側から再定義する立場。ここが、他の 4E アプローチと違う構えの中心になる。

### 動的統合 — 認知は脳でも環境でもなく、その動的カップリングのなかにある

エナクティブ派は Varela / Thompson / Rosch『The Embodied Mind』（1991）に発する立場で、現象学・理論生物学・行動ベースのロボティクス・仏教哲学の交差点に立つ。

拡張派と共有するのは、認知は脳の一点ではなく、**脳・身体・環境の統合系に分散している**という見方だ。ここまでは Clark と同じ語彙を借りる。だが分かれ目はここから先にある。エナクティブ派は機能主義を退け、**生物学的身体の固有性が認知の構成に「還元不可能・代替不可能」な役割を果たす**と主張する。Clark なら鉛筆とノートと神経が機能的に同じ役を果たせるが、エナクティブ派はそれを譲らない。代謝・情動・心拍・呼吸を備えた「生きている身体」が、認知の構成要素として絶対に外せない。

センサリーモーター・エナクティビズム（Noë / O'Regan / Hurley）はこの構えを具体化する。脳内の表象に代わって据えるのは、「**センサリーモーター随伴性**」と「**アフォーダンス**」だ。頭を動かせば視野内の対象位置が法則的に変わる——その随伴性に対する身体的な調律こそが、知覚を**構成する**のだという。

ここで vol 02 で保留にした問いがひとつ回収される。Gibson のアフォーダンスは vol 02 では埋め込みの章で鍵概念として登場した。だがその源流の力学が本気で展開されるのは、書き手の読みとしては**このエナクティブ章こそが「本舞台」**だ。アフォーダンスは「環境が動物に提供する可能性」だが、それが立ち上がるためには**動物側の身体的調律**が要る。両者がカップルする場——そこが知覚の住み処になる。

### 表象を撤去する — エナクティブの中核的な賭け

エナクティブ派の中心的な賭けはここにある。**「もはや何も表象させない」**という選択だ。

CC は「心は脳内の表象を計算的に操作する」と考える。これに対して、拡張派の Clark は表象の範囲を脳の外へ広げたが、表象そのものは「**行為志向表象（AOR）**」という最小限の形で残した。エナクティブ派は、この最小限の表象すら拒否する。

論証の根幹を担うのが Hutto & Myin の「**内容のハード問題**（the hard problem of content）」だ。問いはシンプルで強い：「**乗り物（vehicle）が、意味のある内容を、どうやって担うのか**」。脳の物理状態がなぜ世界の何かを「指す」ことになるのか、その橋渡しが説明できない以上、表象を持ち出す説明上の利得はないのではないか。

そこでエナクティブ派は語彙そのものを差し替える。「表象 / 内容 / 情報処理 / 計算的符号化」を捨て、「**調律（attunement）／ 身体的関与 ／ 動的カップリング ／ センサリーモーター随伴性 ／ アフォーダンス**」を据える。脳は「外界をテストするためのサンプリング装置」ではなく、身体・環境を含む全体系の中の一つの複雑な回路になる。Clark が言う「能動的推論（active inference）」も、エナクティブの語彙では「**推論**」ではなく「**調律された行為的関与**」と読み替えられる。

メルロ＝ポンティの一文がここで効いてくる。「私の身体は『表象』を経由せず、『象徴的・対象化的機能』に従属することなく、その世界を理解している」。これは飾りの引用ではない。冒頭で引いた**系譜の違い**——エナクティブ派は現象学を認知科学に持ち込んでいる——が、この一文の重みとして可視化される。

### 間主観性とセンス・メイキング — 意味は相互作用に生成する

エナクティブ派にとって、他者はオプションではない。知覚にも、記憶にも、行為にも、問題解決にも、最初から織り込まれている。

赤ちゃんは生後早期から、表情・姿勢・身振り・声のイントネーションを通じて大人と**相互調律**を行う（Trevarthen の「一次的間主観性」）。鏡像ニューロンは「シミュレーション装置」ではなく、他者の運動意図の知覚に伴う準備応答として読み替えられる。標準的な Theory of Mind が「3歳児は誤信念課題に失敗する」と説明してきた現象も、実験設定の特殊性として読み直される——3歳児を相互作用に参加させた途端、正答率は大幅に上がる。

ここから De Jaegher & Di Paolo の「**参与的センス・メイキング（participatory sense-making）**」が立ち上がる。**意味は個人の中にあるのではない**。相互作用そのものの動的展開と、参加者の相互変容の重なり目に**生成する**ものだ。

これはデザインに直接効く読み替えだと思う。「意味のあるオブジェクトを設計する」のではなく、「**意味が立ち上がる相互作用そのものを設計する**」へ。サービスもプロダクトも UI も、ユーザーと環境が動的にカップルする場として捉え直せる。vol 02 で「動きの中で意味が立ち上がる場の設計」と書いた読み替えを、エナクティブはもう一段精密化してくれる。

### 創造性のひっかかり — Clark 本人が認めた壁

第7章は技術的になるが、デザイン側から見ると押さえどころは絞れる。自由エネルギー原理とエナクションの緊張関係、その内側に潜む**創造性**の問題だ。

自由エネルギー原理は Karl Friston が立てた原理で、「生体は自由エネルギー（≒驚き）を最小化するように能動的推論を行う」と定式化する。Clark 派はこれをエナクティブと自然に接続できると論じてきた。

ところが Di Paolo / Thompson / Beer（2022）が痛い場所を突く。エナクティブな生命にとって決定的に重要な「**歴史性（historicity）**」——発達・可塑性・スキル獲得・習慣形成・創造性——を、自由エネルギー原理の予測誤差最小化モデルは取り逃がす、と。そして議論のピークは創造性で訪れる。「驚きを最小化する」原理は、新規性の追求・遊び・芸術・ユーモアをどう説明するのか。ここで Clark 本人が——**予測的処理の最強の擁護者である Clark 本人が**——「予測的処理は創造的・即興的行為のよい説明にはならない」と認めた。

エナクティブ派はこの空白を、別の構えで拾いに行く。ギャラガーが引くパフォーマー——演奏家や舞踏家——のモデルがその輪郭をくっきりさせる。彼らは訓練されたスキルを土台にしながら、「**次に何が起こるか予測しないこと**」を引き受ける。脳・身体・環境がそれぞれの場で、予測不能なかたちでカップルすることを可能にする。自動性の対極にある、「**ヒートでフレキシブルな習慣**」だ。

冒頭で引いた Clark という foil の手応えが、ここでもう一度効いてくる。Clark が最後の壁の手前で立ち止まったその場所には、「予測されえないものをどう生きるか」という問いが残されていた。エナクティブ派はその問いを、機能主義の壁の向こうへ持っていく。

### エナクティブが広げる地形

ギャラガーが第8章で挙げる応用領域は広い——パフォーマンス研究（音楽・ダンス・演劇）、教育（VR・全身没入型科学学習）、文学・人文学、**建築**（身体化された行為と空間設計の関係）、経済学（認知制度）、精神医学・臨床推論、医療・理学療法。デザイン野帳の関心からは、建築・パフォーマンス・教育に目が向く。

エナクティブは、認知を「脳の計算」から「**生きていることの一様態**」へと描き直そうとする試みだ。表象を捨てる、機能主義を退ける、語彙を差し替える——どれも奇抜なジェスチャーではなく、その描き直しが本気で要請する作業だった。

冒頭の補助線でもう一度確認すれば、Clark は「身体も環境も道具も認知の一部だ」というところまで来て、機能主義の手前で立ち止まった。エナクティブ派はそこから一歩踏み込んで、認知を生きている系の動的な統合として描き直した。原典のヴァレラまで降りてみたら、その描き直しはまた違う景色で見えてくる気がする。
